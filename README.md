# HDL Designs

This repository contains my digital design implementations using Verilog, VHDL, and SystemVerilog.

It focuses on building a strong foundation in hardware description languages through structured designs ranging from basic logic to small functional modules.

---

## 🧠 Focus

- Understanding digital logic using HDL
- Writing clean and synthesizable code
- Exploring multiple HDLs (Verilog, VHDL, SystemVerilog)
- Building step-by-step design knowledge from basic to sequential systems
